 
  🚦 Traffic Light Controller using RTL (VLSI)
📌 Repository Description

A Traffic Light Controller designed using RTL (Register Transfer Level) in VLSI. This project implements a Finite State Machine (FSM) to control traffic signals (Red, Yellow, Green) with proper timing logic using Verilog/VHDL.

📖 Description

This project demonstrates the design of a traffic light control system using RTL methodology. The system controls the sequence of traffic signals automatically based on clock-driven state transitions. The design is simulated, synthesized, and implemented using standard VLSI tools.

🧠 Objective

Design a traffic light system using RTL

Implement FSM-based sequential logic

Perform simulation, synthesis, and implementation

Verify the design functionality

⚙️ Working

The system follows a continuous cycle:

🟢 Green Light ON – Vehicles move

🟡 Yellow Light ON – Slow down

🔴 Red Light ON – Stop

The transitions are controlled using a clock signal and state machine logic.

🧩 Design Approach

🔹 Based on Finite State Machine (FSM)

🔹 Each signal represents a state

🔹 State transitions occur using clock pulses

🔹 Implemented using Verilog/VHDL RTL design
